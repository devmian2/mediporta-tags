# TagsTable

Reusable tags table component responsible for display tags and their respective usage count.
Style customization is possible through `sx` prop. It accepts `tags` prop in `[{ id: string, content: string, count: number }]` format.
Additionaly it is possible to pass custom columns width through `options` prop.
